## Kurulum ⬇️

Github'dan kaynak kodu indirin 💾
`git clone https://github.com/DogukanUrker/flaskBlog.git`

dizine git 📁
`cd flaskBlog`

gereksinimleri yükle 🔽
`pip install -r requirements.txt`

çalışmaya hazır 🎉
`python app.py`
veya
`python desktop.py`
flaskBlog'u bir masaüstü uygulaması olarak çalıştır 💯
